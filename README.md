## Description

Game Hub website is selling new and used videogames for different platforms.

## Built with

- HTML
- CSS
- Javascript

## Getting Started

### Installing

1. Clone the repo:

```bash
git clone github.com/Noroff-FEU-Assignments/cross-course-project-corrado1982.git
```

2. Install the dependencies:

```
npm install
```

### Running

```bash
npm run start
```
